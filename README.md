# AI Systems Engineer — Career Transition Roadmap

A public log of my path from full-stack software engineer to AI systems engineer.
Built and published one phase at a time as I work through it.

> Each phase folder appears here when I actually begin it — I'm not dropping the whole map up front. Boxes get ticked as I finish things. Notes and code I produce live inside each phase folder; the matching blog/LinkedIn posts get linked from `posts.md` inside the folder.
